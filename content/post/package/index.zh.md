## 读音
/pækɪdʒ/

## 释义
- **名词 n**:
  - 包裹，包装：A container, typically a box or wrapper, used for sending or storing goods.
  - 软件包，程序包：A collection of software components grouped together for distribution or installation.

- **动词 v**:
  - 打包，封装：To put things into a package, especially for transport or sale.
  - 制作软件包：To create a software package for distribution.

## 短语
- "package deal"：一套交易，一揽子计划，通常包括多个相关项目。
- "package tour"：打包旅行，由旅行社安排的包括交通、住宿和观光在内的全套旅游服务。

## 例句
- 名词: "I ordered a new phone online and it arrived in a small package."
- 动词: "The software developer needs to package the application before releasing it."
