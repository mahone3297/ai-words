## 读音
/strɪŋ/

## 释义
- **名词 (n)**: 1. 由一个或多个字符组成的序列，通常在计算机科学中指文本或数据的连续体。例如: "The program read the user's input from a string."
   2. 细长而柔韧的线或绳。例如: "She tied the package with a thin string."
- **动词 (v)**: 将物体串联起来。例如: "He strung the beads together to make a necklace."

## 短语
- "string along" 意为欺骗或拖延某人。例如: "Don't let him string you along; he has no intention of helping."
- "string of pearls" 可以指一串珍珠项链，或者一系列连续的事物。例如: "She wore a beautiful string of pearls around her neck."

## 例句
- 名词: "The computer program used a string to store the user's name."
- 动词: "She carefully strung the Christmas lights around the tree."

- line
- series
- thread
