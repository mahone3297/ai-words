## 读音
/məˈmɔri/

## 释义
- **名词**: 记忆；回忆。例如: "He has a great memory for names and faces."
- **名词**: 存储器（计算机科学术语）。例如: "The computer's RAM is its short-term memory."

## 短语
- "Store something in memory" 意为将信息存储在记忆中。
- "Loss of memory" 指的是记忆力减退或丧失。
- "From memory" 表示凭记忆，不借助外部帮助。例如: "I can recite the poem from memory."

## 例句
- 名词: "Her memory of the event is still vivid after all these years."
- 名词（计算机科学）: "The new laptop has a large memory capacity for storing files."
