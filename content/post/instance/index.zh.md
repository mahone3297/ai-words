## 读音
/ˈɪnstəns/

## 释义
- **名词**:
  - 例子: "An instance of a problem can be a specific situation where it occurs." (一个问题的一个实例可以是它发生的具体情况)
- **形容词**:
  - 例子: "This is an instance when the rule doesn't apply." (这是一个规则不适用的例子)

## 短语
- "For instance" 用来举例说明，例如: "There are many benefits, for instance, improved health and increased productivity."
- "In some instances" 表示在某些情况下，例如: "In some instances, the software may crash."

## 例句
- 名词: "The program crashed due to a particular instance of a bug."
- 形容词: "This is an instance where the old method proves more efficient than the new one."

## 同义词
- Example
- Case
- Situation
- Occurrence
- Instance (作为名词时，同义词也包括自身)
