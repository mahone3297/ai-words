## 读音
/ˈlaɪbrəri/

## 释义
- **名词**:
   - 公共或私人收藏书籍、期刊、多媒体资料的地方，供阅读、研究或借阅。例如: "The library has an extensive collection of ancient texts."
   - 一个软件或计算机系统中包含多种功能模块或代码的地方。例如: "Python's standard library provides many useful modules for developers."

## 短语
- "City Library" 城市图书馆
- "Digital Library" 数字图书馆
- "Library Card" 借书卡

## 例句
- 名词: "I spent hours in the library researching for my history project."
- 名词（计算机相关）: "We need to import the math library to use the square root function in our code."
