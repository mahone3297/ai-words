## 读音
/run/

## 释义
- **动词**:
  - 跑，快速移动（脚离地并交替着地）。例如: "The child was running happily in the park."
  - 经营，管理。例如: "She runs a successful bakery."
  - 执行，操作（计算机程序或机器）。例如: "I need to run a virus scan on my computer."
  - 持续，继续。例如: "The show has been running for five seasons."

- **名词**:
  - 跑步的动作或过程。例如: "He went for a morning run along the beach."
  - 连续的一段时间。例如: "They had a winning run in the tournament."
  - （液体）流动。例如: "Water was running down the drain."

## 短语
- "Run for office" 意为竞选公职。
- "On the run" 表示在逃，逃跑。
- "In a run" 指处于忙碌或连续的状态。

## 例句
- 动词: "The dog ran after the ball when it was thrown."
- 名词: "After a long day at work, he enjoyed his evening run to clear his mind."

## 同义词
- Move quickly: dash, sprint, hurry
- Manage: operate, oversee, administer
- Continue: last, persist, endure
