## 读音
/riːˈkwest/

## 释义
- **名词** (n): 请求；要求；申请。指正式或非正式地提出想要得到某物或希望某事发生。例如: "She made a polite request for more information."
- **动词** (v): 要求；请求。表示向某人提出需要或期望。例如: "He requested an audience with the king."

## 短语
- "to make a request" 请求做某事。
- "request for" 对...的要求，例如: "A request for assistance."

## 例句
- 名词: "The customer submitted a request for a refund."
- 动词: "Could you please request the file from the server?"

## 同义词
- Ask
- Demand
- Inquire
- Petition
- Seek
