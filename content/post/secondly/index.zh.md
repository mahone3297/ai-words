## 读音
/ˈsekəndli/

## 释义
- **副词**: 表示顺序，意为“其次，第二点”。用来引出观点或论据的第二个方面。例如: "Firstly, we need to gather data; secondly, we should analyze it."

## 短语
- "Secondly, let's discuss the budget" 用在列出多个要点时，表示接下来要讲的第二个重点。

## 例句
- "First, we need to set our goals; secondly, we need to create a plan to achieve them."

## 同义词

- Moreover
- Furthermore
