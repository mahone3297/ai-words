## 读音
/ˈəʊnli/

## 释义
- **副词**: 只有，仅仅，单独地。表示限定或强调唯一性。例如: "There is only one solution to this problem."
- **形容词**: 唯一的，仅有的。用来修饰名词，表示独一无二。例如: "This is my only chance."

## 短语
- "not only...but also..." 表示“不仅...而且...”，用于连接两个并列成分。
- "the only way" 指的是唯一的方法或途径。

## 例句
- 副词: "I have only a few minutes before I need to leave."
- 形容词: "This is the only key to the safe."
