## 读音
/ˈɑːbʤɛkt/

## 释义
- **名词 (n)**: 物体；客体；目标；主题。例如: "The painting depicts a still life with various objects on a table."
- **名词 (n)**: 计算机科学中，对象是指具有特定属性和方法的数据结构。例如: "In programming, an object is an instance of a class."
- **名词 (n)**: 逻辑或哲学中，指被认识或感知的事物。例如: "The object of his desire was the new car."
- **宾语 (n)**: 句子中动词所作用的人或事物。例如: "She loves him, where 'him' is the object of the verb 'loves'."

## 短语
- "The main object of the game is to win." 游戏的主要目标是获胜。
- "He is studying computer science with the object of becoming a software developer." 他学习计算机科学的目标是成为一名软件开发者。

## 例句
- 名词: "The lost object was found under the sofa."
- 宾语: "She saw a cat running across the street, where 'cat' is the object of the verb 'saw'."
