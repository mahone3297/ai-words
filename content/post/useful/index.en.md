## Definitions
### Adjective
- **Meaning**: Having practical value or usefulness; helpful or beneficial. For example: "The old shed still had some useful tools inside."
- **Antonym**: Useless

### Noun (informal)
- **Meaning**: Something that is useful or helpful. For example: "This app provides a lot of useful information."

## Phrases
- "Be useful" implies being helpful or serving a purpose.
- "A useful skill" refers to a skill that is beneficial.
- "A useful tip" is a piece of advice that is helpful.
